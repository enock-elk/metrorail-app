01. Overview

Say Goodbye to Waiting.

A Progressive Web App (PWA) designed for Gauteng commuters. It provides real-time countdowns, offline schedules, and community-driven alerts for the Metrorail network.

  • 100% Offline Capability
  • Privacy-First Geolocation
  • Works on any device (Web/Mobile)

02. Architecture

Built for Speed & Scale

🎨

Frontend

Vanilla JavaScript + Tailwind CSS. Single-file architecture for instant loading.

🔥

Backend

Firebase Realtime DB (Schedules) + Firestore (Alerts) + Auth (Anonymous).

📊

CMS Pipeline

Google Sheets -> Google Apps Script -> Firebase. Allows easy manual data entry.

03. Latest Updates (V3.17)

Community Powered

⚠️

Crowdsourced Alerts

Users can now report delays, cancellations, or safety issues directly in the app. These reports appear in a real-time feed for all other users.

📍

Auto-Locate on Load

Logic updated to automatically detect GPS permission. If granted, the app instantly selects the nearest station without user input.

🔐

Robust Authentication

Implemented Anonymous Auth with persistence fallback (Local Storage -> In-Memory) to solve browser/iframe restrictions.
